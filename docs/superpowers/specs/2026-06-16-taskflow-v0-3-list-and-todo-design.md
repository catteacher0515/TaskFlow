# TaskFlow v0.3 列表与 Todo 交互规格

日期：2026-06-16

## 范围

v0.3 只处理两件事：

1. 项目列表按状态分组并支持折叠。
2. 通用任务改为 Todo 交互，支持完成音效、右键菜单和反馈撤销。

模板管理暂不修改。

## 项目列表

- 项目列表按状态分组：进行中、待开始、暂停、已完成。
- 所有分组都可以折叠。
- 进行中默认展开。
- 已完成默认折叠。
- 分组标题显示数量。
- 已完成项目仍可展开后进入详情。
- 项目完成前记录完成前状态。
- 已完成项目支持恢复到完成前状态。
- 恢复动作不产生反馈。

## 通用任务 Todo 交互

- 通用任务节点左侧显示 checkbox。
- 勾选 checkbox 将任务标记为完成。
- 完成时播放短音效。
- 已完成任务视觉上弱化并显示勾选状态。
- 不做了不再显示为常驻按钮。
- 任务节点支持右键菜单。
- 右键菜单至少包含：
  - 不做了。
  - 恢复为待办。
- 恢复为待办表示从完成、不做了或未处理恢复到未开始。
- 恢复为待办会撤销该任务之前产生的有效反馈。
- 恢复动作本身不产生反馈。

## 反馈撤销

- 任务完成或不做了时产生的反馈必须可撤销。
- 撤销后，当前面板反馈数、最近反馈和反馈页都不再计入该反馈。
- 撤销不删除原始日志行，而是把对应反馈标记为无效，保留本地历史可追踪性。

## 验收标准

- 项目列表显示可折叠的进行中和已完成分组。
- 进行中默认展开，已完成默认折叠。
- 已完成项目详情中显示恢复按钮。
- 恢复后项目回到完成前状态。
- 通用任务可以用 checkbox 完成。
- checkbox 完成会调用任务状态接口并播放音效。
- 右键任务出现菜单，可以选择不做了。
- 已完成或不做了的任务可以恢复为待办。
- 恢复为待办后，对应反馈不再计入反馈总数和反馈页。
